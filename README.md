# Дипломный проект "NeWork"

### по специальности Android developer
---

### Описание проекта

#В рамках дипломного проекта разработано функционирующее приложение по типу LinkedIn.

### Используемые инструменты:

- MVVM архитектура
- используемые библиотеки:
    - Material Design
    - Coroutines
    - Room
    - Paging
    - Hilt
    - Flow & LiveData
    - MapKit(Yandex Maps)
    - Retrofit
    - Media3
